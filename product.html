Описание

Защитный чехол TOPMOUNT для охотничьего ружья, карабина

Для использования при транспортировке и перемещения оружия с установленным оптическим прицелом и ДТК на ходовой охоте. Использование с погонным ремнём карабина. Применим при габаритной длине ружья от 90 до 130 см. Непромокаемый. Модель с ВСТАВКАМИ ОРАНЖЕВОГО ЦВЕТА!

Мгновенное снятие с карабина для быстрого применения оружия. Фиксация на оружии при помощи вшитого резинового жгута.

Для зачехленния:

  1. надеть край чехла на ствол
  2. взявшись за противоположный край, натянуть на ружьё
  3. зацепить край чехла за пятку приклада
  4. натяжение вшитого жгута надёжно зафиксирует чехол на карабине
  5. использовать для переноски штатный погонник

Для снятия, потяните за петлю расположенную на чехле со стороны приклада и высвободите его. Резиновый жгут начнёт сжиматься помогая освободить оружие от чехла. Сдёрните чехол со ствола простым движением вверх. Это можно сделать мгновенно для быстрого применения охотничьего оружия.
